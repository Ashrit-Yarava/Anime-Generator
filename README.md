# Waifu Generator

**Generate anime faces with DCGANs**

### Generator Architecture
```
RecursiveScriptModule(
  original_name=Module
  (l1): RecursiveScriptModule(
      original_name=Module
          (0): RecursiveScriptModule(original_name=Module)
	  )
  (conv_blocks): RecursiveScriptModule(
      original_name=Module
	(0): RecursiveScriptModule(original_name=Module)
	(1): RecursiveScriptModule(original_name=Module)
	(2): RecursiveScriptModule(original_name=Module)
	(3): RecursiveScriptModule(original_name=Module)
	(4): RecursiveScriptModule(original_name=Module)
	(5): RecursiveScriptModule(original_name=Module)
	(6): RecursiveScriptModule(original_name=Module)
	(7): RecursiveScriptModule(original_name=Module)
	(8): RecursiveScriptModule(original_name=Module)
	(9): RecursiveScriptModule(original_name=Module)
	(10): RecursiveScriptModule(original_name=Module)					    
	)
	  
)
```

### Using the Application

Run the program. In the current directory, there will be a .png image that is the image.
> THE IMAGE WILL BE DELETED UPON CLOSING THE APPLICATION!
